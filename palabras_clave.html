PALABRAS CLAVE UTILES¬(EXTRAS)



pass:

Sirve para pasar de un problema o para saltar el problema.

Ejemplo sin la palabra pass:

x = [1,2,3]

for variable in x:
-----#Comentario

print("Fin del programa")


Resultado:

¡¡ERROR!!

Esto nos dará un error ya qu een el for no colocamos instruccion, ahora veremos el ejemplo con la palabra clave pass.




Ejemplo con la palabra clave pass:


x = [1,2,3]

for variable in x:
-----#Comentario

-----pass

print("Fin del programa")


Resultado:

Fin del programa


Como vemos nos salta el error y sigue con el resto del programa.




continue:

Se usa para saltar letras o palabras, o que no quieras que aparezcan en pantalla.


Ejemplo:

x = "Python"

for letras in x:

----if letras == "y":

-----continue

----print(letras)


Resultado:

Pthon


Como vemos se saltó la letra (y) y continuó con el resto de las letras.




break:

Se usa para parar o romper las cadenas o el bloque de código donde quieras que se detenga.


Ejemplo:

x = "Python"

for letras in x:

----if letras == "y":

-----break

----print(letras)


Resultado:

P


Como vemos se detuvo en la letra (y), osea que no se mostró en pantalla el texto completo.
