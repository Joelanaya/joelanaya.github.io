CONDICIONALES IF, ELIF Y ELSE



QUE SON LAS CONDICIONALES?

Usamos las condicionales para controlar el flujo de nuestro programa, usualmente queremos que cierto código sea ejecutado usando una condición particular ocurra.


Para controlar el flujo de lógica usamos palabras clave:

# if

# elif

# else


#Sitaxis de una condicion if:

if alguna_condicion:

------- colocamos funcion que queramos :)


# Sintaxis para una condicion if/else:

if alguna_condicion:

---------- ejecutar algo

else:

---Aplicar algo mas


# Sintaxis de una condicion if/elif/else:

if alguna_condicion:
--------ejecutar algo

elif:
---ejecutar algo distinto

else:
----hacer algo mas



Ejemplo:


hambre = True

sed = True

if hambre and not sed:

------print("Tenemos hambre!")

elif hambre == True and sed == True:

-------print("Tenemos hambre y sed")

else:
---print("Estamos llenos!")


Tenemos hambre y sed


Y así se utilizan los condicionales, como podemos ver las dos variales son verdaderas por lo tanto el resultado es tenemos hambre.
