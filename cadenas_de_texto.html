CADENAS DE TEXTO



QUE SON LAS CADENAS DE TEXTO?

Las cadenas de texto son secuencias de caracter, usan sintaxis con comillas simples o comillas dobles.

  • 'Hola'


  • "Hola"


  • caracter = "cadena de texto con muchos caracteres"



  • Debido a que las cadenas de texto son secuencias ordenadas significa que podemos usar indexado y slicing para agarrar sub-secciones de una cadena.



Notación de indexado:

Te permite agarrar un caracter singular de una cadena de texto.




Slicing:

Te permite agarrar una subsección de multiples caracteres, un "Slice" de una cadana




  • Tiene la siguiente sintaxis:


  • [start:stop:step] ---> [0:4:2]



  • Start: Es un indice númerico para el slice iniciar.


  • Stop: Es el número donde paramos el slice.


  • Step: Son los pasos que damos en el slice


INDEXADO:

mi_cadena = ("Hola Mundo")

print(mi_cadena[2])

l


mi_cadena = ("Hola Mundo")

print(mi_cadena[3])

a


mi_cadena = ("Hola Mundo")

print(mi_cadena[-1])

o


Bien como podemos observar así se hace el indexado en las cadenas de texto, ahora veamos el slicing :)



SLICING:

mi_cadena = ("hola mundo")

print(mi_cadena[0:4])

hola


mi_cadena = ("hola mundo")

print(mi_cadena[0:7:2])

hl u




PROPIEDADES Y METODOS DE CADENAS

x = "hola mundo"

x = x.upper()

print(x)

HOLA MUNDO

Lo que hace este método es que coloca todas las letras en mayúsculas así el usuario las coloque en minúsculas.


x = "HOLA MUNDO"

x = x.lower()

print(x)

hola mundo

Lo que hace este método es que coloca todas las letras en minúsculas así el usuario las coloque en mayúsculas.


x = "hola mundo"

x = x.split()

print(x)

["hola","mundo"]

Lo que hace este método es que separa las palabras.
