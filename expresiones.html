Lambda:

La función lambda se utiliza para hacer mas funciones o cuando utilizamos mas funciones.


Ejemplo:

numeros = [1,2,3,4,5,6,7]

raiz_cuadrada = lambda num: num**2

print(raiz_cuadrada(5))


Resultado:

25
