VARIABLES



QUE SON LAS VARIABLES?

Las variables son un espacio en memoria en la cual podemos almacenar números,cadenas de texto, booleanos y todo tipo de dato que sea de python.

En la clase pasada trabajamos con números pero es dificil saber que representan estos números si no les asignamos una variable.

Sería bueno asignar a estos tipos de datos un nombre para poder reconocerlos.

EJEMPLO:

mis_casas = 2



REGLAS PARA NOMBRAR VARIABLES:

  • Se considera buena practica usar nombres en minúsculas.

  • Evita usar palabras con significado especial como "list" o "str".

PYTHON USA NOMENCLATURA DINÁMICA

  • Significa que puedes re-asignar variables a otros tipos de datos.

  • Esto hace python bastante flexible al momento de asignar variables.


mis_libros = 2

mis_libros = ["libro1", "libro2"]

Esto es correcto en python :)
