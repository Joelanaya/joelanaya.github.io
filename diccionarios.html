DICCIONARIOS



  • Mapeos desordenados para guardar objetos.


  • Previamente vimos como las listas guardan objetos en una secuencia ordenada.


  • Los diccionarios utilizan un orden basado en par Clave/Valor.


  • Este par Clave/Valor permite al usuario agarrar objetos sin necesidad de saber la locación (número) del indice.


  • Los diccionarios usan llaves {} y dos puntos (:) para simbolizar las llaves y su valor asociado.


  • Cuando debemos escoger una lista o un diccionario?:


  • Diccionario: Objetos retornados por llave, desordenado y no se guarda, se utiliza para cuando no sepas donde se encuentra un objeto.


  • Listas: Objetos retornados por locación, puede ser indexado o slicing.



mi_diccionario = {"llave1":"valor1","llave2":"valor2"}

print(mi_diccionario)

{"llave1":"valor1","llave2":"valor2"}


mi_diccionario = {"llave1":"valor1","llave2":"valor2"}

print(mi_diccionario["llave1"])

valor1
