FUNCIONES ANIDADAS



Las funciones anidadas son funciones dentro de otras funciones

Ejemplo:

name = "Variable global"

def saludo():

---name = "Joel"

---def hola():

-------print("Hola " + name)

---hola()

saludo()


Resultado:


Hola Joel


y bien así son las funciones anidadas
