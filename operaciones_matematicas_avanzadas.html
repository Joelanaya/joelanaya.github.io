MODULUS

Modulus lo que hace es que retorna la sobra de la división, ver si el número es divisible, o si es par o impar, para ello utilizamos este signo (%).



RESTO O SOBRA:


a = 7%4
print(a)
3

Como vemos, la sobra es 3.



DIVISIBLE:


b = 50%5
print(b)
0

Cuando termina en el numero 0, significa que es divisible, de lo contrario aparecerá el número 1.



PAR:


c = 20%2
print(c)
0

Cuando termina en número 0 el vakor es par, pero si termina en número 1 el valor es impar.

POTENCIA


La potencia de cada número, lo que hace es que eleva o multiplica 2 veces el mismo número, para ello utilizamoz dos veces el signo de multiplicación (**):

a = 20**2
print(a)
400

Como podemos observar el número 20 se multiplicó 2 veces


b = 10**2
print(b)
100

Lo mismo sucede con el número 10.



BETMAS



Betmas se usa para hacer operaciones matematicas mas complejas, este conciste en resolver la operacion que se encuentra dentro de parentesis (), y luego resuelve lo que está afuera, veamos un claro ejemplo:


a = (2+10)*(10+4)
print(a)
168

Como podemos observar primero sumó lo que esta dentro de los parentesis, luego multiplicó y luego volvió a sumar.
