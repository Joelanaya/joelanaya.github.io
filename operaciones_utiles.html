range:

Sirve para colocar un rango a los números, o palabras que queramos.


Ejemplo:

for num in range (10):

------print(num)


Resultado:


1
2
3
4
5
6
7
8
9


Como vemos se muestra solo nueve números esto sucede porque python cuenta desde el número cero, si queremos que aparezca el número diez debemos colocar el número once en range.




input:

Sirve para las entradas que ingrese el usuario desde el teclado.

Ejemplo:

texto = input("Escribe un texto aquí: ")
print(texto)

Resultado:

Escribe un texto aquí: Hola, estoy aprendiendo python

Hola, estoy aprendiendo python


Como vemos el programa espera a que ingresemos un texto y luego el texto que colocamos lo imprime en pantalla.
