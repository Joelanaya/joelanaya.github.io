CICLOS WHILE



QUE SON LOS CICLOS WHILE?

Los ciclos while son aquellos que van a continuar ejecutando un bloque de código while (mientras) una condición siga siendo verdadera.

Sintaxis para ciclo while:

while condicion_booleana:

----# Hacer algo



Puedes combinar con else:

while condicion_booleana:

-----# Hacer algo

else:

--# Hacer algo mas





Ejemplo con while:

x = 0

while x < 5:

----- print(f "El valor actual de x es {x}")

x = x+1


Resultado:

El valor de x es 0
El valor de x es 1
El valor de x es 2
El valor de x es 3
El valor de x es 4


Como vemos en el print de while, primero colocamos la letra (f), y despues colocamos la letra {x}, a este metodo se le conoce como f-string, este metodo lo que hace es concatenar o unir texto y variables o números, te recomiendo que uses este metodo para que no te confundas, luego en el resultado vemos que solo salen cuatro numeros si habiamos colocado 5, la respuesta es que python cuenta desde el numero cero y no aparece el número 5, si queremos que salga el número 5 debemos colocar un 6 en la funcion.




Ejemplo con while y else:

x = 0

while x < 5:

----- print(f "El valor actual de x es {x}")

x = x+1

else:

---print("El ciclo terminó")


Resultado:

El valor de x es 0
El valor de x es 1
El valor de x es 2
El valor de x es 3
El valor de x es 4
El ciclo terminó


Como vemos cuando el valor llega al número 5 se detiene y dice que el ciclo ha terminado.
